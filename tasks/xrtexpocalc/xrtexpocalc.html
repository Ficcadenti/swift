NAME

xrtexpocalc -- Generate the exposure map from the sky coordinates instrument map.

USAGE

xrtexpocalc infile outfile [parameter=value ... ]

DESCRIPTION

The task generates the exposure map for XRT Photon Counting mode starting from the sum of the instrument maps in sky coordinates. See 'xrtexpomap' for more details.

PARAMETERS

infile [file name]
Name of the input sky coordinates instrument map FITS file.

outfile [file name]
Name of output Exposure map FITS file.

(history=yes) [boolean]
If set to yes, write parameter values and other information in HISTORY blocks.

(clobber=no) [boolean]
If 'clobber'=yes and outfile=filename, the file with the same name will be overwritten, if it exists.

(chatter = 2) [integer]
Chatter Level (min=0, max=5).
