NAME

xrtimage2cnt -- Generate a sky coordinates counts image from Level 2 Imaging Mode files.

USAGE

xrtimage2cnt infile outfile [parameter=value ...]

DESCRIPTION

The task creates counts images starting from Level 2 Imaging mode files.
Swift XRT Imaging Mode data are stored in image extensions of a FITS file each one containing the DN values of the pixels exceeding the Lower Level Discriminator threshold for a single frame and their position in RAW coordinates. After the processing of the Imaging mode files and the RAW to SKY coordinate transformation it is possible to merge data from different frames using this task to generate a counts image in sky coordinates that can be used for the spatial analysis.

PARAMETERS

infile [file name]
The input FITS Imaging mode file name.

countfile [file name]
The output FITS file name. DEFAULT to use standard naming convention.

(history=yes) [boolean]
Write parameter values and other information in HISTORY blocks.

(clobber=no) [boolean]
If 'clobber'= yes a output file with the same name will be overwritten if it exists.

(chatter = 2) [integer]
Verbosity Level (from 0 to 5).

EXAMPLES

  1. ft> xrtimage2cnt sw00073149002xim00praw.img.gz DEFAULT chatter=5
