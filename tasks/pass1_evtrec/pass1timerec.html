NAME

xrttimerec --

USAGE

xrttimerec infile [parameter=value ... ]

DESCRIPTION

PARAMETERS

infile [file name]
The FITS file name and extension from which the photon counting PHAS column must be read.

outfile [file name]
The new FITS file with the PHA, GRADE and PixsAbove columns filled. If 'NONE' is input, the input file will be overwritten.

(split = 40) [integer]
The split threshold level desired. If set to a negative value the on board split threshold (read form the input file keyword 'SPLITTHR') is used.

(history=yes) [boolean]
Write parameter values and other information in HISTORY blocks.

(clobber=no) [boolean]
Overwrite the output file.

(chatter = 2) [integer]
Verbosity Level from 0 to 5
